# 라이더 스타터 AI 기획 정리

## 1. 프로젝트 개요

### 프로젝트명
**라이더 스타터 AI**

### 한 줄 설명
초보 배달 라이더가 자신의 예산, 운행 경험, 배달 지속 계획에 따라 현실적인 오토바이를 선택하고, 배달 입문에 필요한 기본 가이드를 받을 수 있도록 돕는 AI 기반 추천 시스템이다.

### 핵심 목적
배달을 처음 시작하는 초보 라이더가 다음과 같은 문제를 쉽게 해결하도록 돕는다.

- 어떤 오토바이를 사야 하는지 모름
- 초기 비용이 얼마나 드는지 모름
- PCX, NMAX, UHR, VF100R 중 무엇이 맞는지 판단하기 어려움
- 중고 오토바이 구매 시 무엇을 확인해야 하는지 모름
- 배달 시작 전 준비물, 면허, 보험, 안전수칙을 잘 모름
- 주변 경험자에게 계속 물어봐야 하는 반복 질문이 많음

---

## 2. 프로젝트 방향이 정해지기까지의 판단

초기에는 아버님이 배민플러스/배달대행 관련 라이더 관리를 하고 있으므로, 라이더 운영 보조 시스템을 생각했다.

하지만 대화 과정에서 아래 기능들은 현실성이 낮거나 이미 배민 시스템에 있거나, 아버님 운영 방식과 맞지 않아 제외했다.

### 제외한 아이디어

| 제외 아이디어 | 제외 이유 |
|---|---|
| 피크타임 라이더 필요 인원 예측 | 라이더들이 자유롭게 나오고 안 나오는 구조라 관리자가 강제 배치할 수 없음 |
| 인수인계 시스템 | 운영을 아버님이 혼자 하고 있어 필요성이 낮음 |
| 민원/문의 대응 AI | 아버님이 사람 상대를 잘하고 실제 민원이 거의 없음 |
| 사고 긴급도 분류 AI | 사고 순간에는 라이더가 차분히 입력할 여유가 없고, 전화가 더 안전할 수 있음 |
| 장비 대여/반납 관리 | 실제로 장비 대여를 하지 않음 |
| 날씨 공지 AI | 카카오톡 오픈채팅방에 간단히 올리면 끝이라 별도 시스템 필요성이 낮음 |
| 라이더 방문 기록/활동 패턴 분석 | 서로 비슷한 기능이고 실질적 필요성이 낮음 |
| 라이더 빠른 검색 시스템 | 배민 시스템에 이미 있음 |
| 정산 예상 계산기 | 배민 시스템에 이미 있음 |
| 신규 라이더 모집/등록 관리 | 현재 라이더는 거의 모인 상태이고 신규 모집이 목표가 아님 |
| 배달대행 운영 확장 시스템 | 아버님은 돈을 더 쓰기보다 현황 유지가 목표임 |

최종적으로는 배민 운영 자체보다, **초보 라이더가 처음 배달을 시작할 때 필요한 오토바이 선택과 입문 가이드**에 초점을 맞추기로 했다.

## 3. 최종 프로젝트 컨셉

### 최종 주제
**초보 배달 라이더를 위한 AI 오토바이 추천 및 입문 가이드 시스템**

### 서비스 컨셉
초보 배달 라이더가 본인의 예산, 경험, 운행 계획, 구매 성향을 입력하면 AI가 대표 배달용 오토바이 중 적합한 모델을 추천하고, 추천 이유와 주의사항, 중고 구매 체크리스트, 입문 가이드를 제공한다.

### 핵심 사용자
- 배달 일을 처음 시작하려는 사람
- 오토바이 구매 경험이 부족한 사람
- PCX, NMAX, UHR, VF100R 중 무엇을 사야 할지 고민하는 사람
- 초기 비용이 부담되어 현실적인 선택지를 찾는 사람
- 장기적으로 배달을 할지 아직 확신이 없는 초보자
- 배달용 오토바이 유지비, 세팅비, 구매 기준을 잘 모르는 사람

---

## 4. 다룰 오토바이 범위

현실적으로 모든 오토바이를 다루는 것은 불가능하므로, 대표적인 배달용 기종만 다룬다.

### 대상 모델

| 모델 | 가격 기준 | 배달 세팅 포함 예상 | 포지션 |
|---|---:|---:|---|
| **PCX** | 약 470만 원 | 약 500만 원대 | 가장 대중적, 장기 운행용 |
| **NMAX** | 약 470만 원 | 약 500만 원대 | PCX 대안, 장기 운행용 |
| **UHR 등 기타 스쿠터** | 450만 원 이하 | 약 480만~500만 원 이하 예상 | 중간 가격대 대안 |
| **VF100R** | 약 250만 원 | 약 300만 원대 | 저예산 초보 입문용 |

### 시장 비중 가정

사용자 제공 기준:

| 모델군 | 비중 |
|---|---:|
| PCX | 약 60% |
| NMAX | 약 20% |
| UHR 등 기타 | 약 20% |
| VF100R | 별도 입문용 모델로 취급 |

### 범위 제한 이유
- 초보 라이더가 실제로 많이 고민하는 모델 위주로 구성
- 모델이 너무 많아지면 추천 기준이 흐려짐
- 연식, 중고 상태, 수리 이력까지 모두 다루면 프로젝트 범위가 과도하게 커짐
- 대표 모델만 다뤄도 초보자용 추천 시스템으로 충분히 설득력 있음

## 5. 모델별 포지션 정리

### PCX

#### 추천 대상
- 예산이 500만 원 전후 가능
- 가장 대중적인 모델을 선호
- 오래 배달할 계획이 있음
- 중고 거래, 부품 수급, 정보량을 중요하게 봄
- 주변 라이더들이 많이 타는 모델을 원함

#### 장점
- 배달 시장에서 가장 대중적인 선택지
- 정보가 많고 사례가 많음
- 중고 거래가 비교적 활발함
- 장기 운행을 고려하는 초보자에게 안정적인 선택지

#### 주의사항
- 초기 비용이 높음
- 중고 구매 시 주행거리만 보지 말고 사고 여부, 정비 이력, 소모품 상태 확인 필요

---

### NMAX

#### 추천 대상
- PCX와 비슷한 가격대에서 다른 선택지를 원함
- 장기 운행을 생각함
- PCX 외 대중적인 모델을 비교하고 싶음

#### 장점
- PCX와 함께 많이 비교되는 대중적인 배달용 스쿠터
- 장기 운행 후보로 고려 가능
- PCX 대안으로 선택 가능

#### 주의사항
- 초기 비용은 PCX와 비슷하게 높음
- 주변 정비 접근성, 부품 수급, 중고 매물 상태를 확인해야 함

---

### UHR 등 기타 스쿠터

#### 추천 대상
- PCX/NMAX는 부담되지만 125cc급 스쿠터를 원함
- 450만 원 이하 가격대를 고려함
- 중간 가격대 대안을 찾고 있음
- 주변에서 정비 가능한지 확인할 수 있음

#### 장점
- PCX/NMAX보다 가격 부담이 낮은 대안이 될 수 있음
- 125cc급 배달용 스쿠터 후보로 고려 가능

#### 주의사항
- 초보자는 가격만 보고 고르면 안 됨
- 주변 정비 가능 여부와 부품 수급을 확인해야 함
- 장기 운행 안정성은 PCX/NMAX와 비교 필요

---

### VF100R

#### 추천 대상
- 초기 비용을 줄이고 싶음
- 배달을 오래 할지 아직 확신이 없음
- 300만 원 안쪽 또는 300만 원대 초반으로 시작하고 싶음
- 완전 초보라 고가 스쿠터가 부담됨
- 입문용으로 먼저 경험해보고 싶음

#### 장점
- 오토바이 가격 약 250만 원
- 배달 세팅 포함 약 300만 원대
- PCX/NMAX/UHR보다 초기 비용 부담이 낮음
- 초보자가 배달을 경험해보기 좋은 현실적인 선택지

#### 주의사항
- 장기 운행, 승차감, 적재성, 유지관리 측면에서는 상위 모델과 비교 필요
- 풀타임 장기 배달을 계획한다면 PCX/NMAX급과 비교해야 함

## 6. 추천 기준

### 주요 입력 질문

| 질문 | 선택지 예시 |
|---|---|
| 예산이 얼마인가요? | 300만 이하 / 300~450만 / 450~500만 / 500만 이상 |
| 오토바이 경험이 있나요? | 없음 / 조금 있음 / 오래 타봄 |
| 배달을 얼마나 할 예정인가요? | 잠깐 해보기 / 부업 / 장기적으로 할 예정 |

| 중고 구매도 괜찮나요? | 가능 / 불안함 / 신차 선호 |
| 가장 중요한 기준은 무엇인가요? | 가격 / 대중성 / 유지비 / 장기 운행 / 초보 안정성 |
| 배달 지역은 어떤가요? | 평지 위주 / 언덕 많음 / 장거리 많음 |
| 체격 부담이 있나요? | 가벼운 오토바이 선호 / 상관없음 |

## 8. 추천 점수 로직

처음부터 머신러닝을 쓰기보다, **규칙 기반 추천 + AI 설명 생성** 방식이 현실적이다.

### 추천 점수 예시

```text
추천 점수 =
예산 적합도 35점
+ 초보 적합도 25점
+ 장기 운행 적합도 15점
+ 대중성 15점
+ 유지관리 편의성 10점
```

### 기준 설명

| 기준 | 설명 |
|---|---|
| 예산 적합도 | 사용자의 예산 안에 들어오는가 |
| 초보 적합도 | 초보자가 조작하고 관리하기 부담 없는가 |
| 장기 운행 적합도 | 오래 배달할 때 괜찮은가 |
| 대중성 | 시장에서 많이 쓰이고 정보가 많은가 |
| 유지관리 편의성 | 정비와 부품 수급이 쉬운가 |

## 9. 추천 결과 구조

추천 결과는 단순히 하나만 보여주지 말고, 아래 구조로 제공한다.

```text
1순위 추천 모델
2순위 대안 모델
추천 이유
주의할 점
구매 전 체크리스트
초보자 가이드
```

### 예시 1: 저예산 초보

#### 입력값
```text
예산: 300만 원
경험: 완전 초보
운행 계획: 일단 해보고 결정
중고 구매: 부담됨
목표: 초기 비용 최소화
```

#### 추천 결과
```text
추천 모델: VF100R

추천 이유:
- 배달 세팅 포함 약 300만 원대로 시작 가능
- PCX/NMAX/UHR보다 초기 비용 부담이 낮음
- 완전 초보가 처음 배달을 경험해보기 좋은 선택지

대안:
- 예산을 450만 원까지 올릴 수 있다면 UHR 계열도 고려 가능

주의사항:
- 장시간 배달을 계속할 계획이라면 추후 PCX/NMAX급과 비교 필요
- 구매 전 브레이크, 타이어, 엔진 소음, 서류 상태를 확인해야 함
```

### 예시 2: 장기 운행 초보

#### 입력값
```text
예산: 500만 원
경험: 오토바이 경험 조금 있음
운행 계획: 장기적으로 배달 예정
목표: 대중적인 모델 선호
```

#### 추천 결과
```text
추천 모델: PCX

추천 이유:
- 배달 시장에서 가장 많이 쓰이는 대중 모델
- 정보가 많고 주변 라이더 사례가 많음
- 장기 운행을 생각하는 초보자에게 안정적인 선택지

대안:
- PCX 외 대안을 원한다면 NMAX도 비교 가능

주의사항:
- 초기 비용이 약 500만 원대로 높음
- 중고 구매 시 사고 여부와 정비 이력 확인이 중요함
```

## 10. 핵심 기능

### MVP 필수 기능

| 기능 | 설명 |
|---|---|
| 초보 라이더 진단 | 예산, 경험, 운행 계획, 구매 성향 입력 |
| 오토바이 추천 | PCX/NMAX/UHR/VF100R 중 추천 |
| 추천 이유 설명 | 왜 이 모델이 맞는지 쉬운 말로 설명 |
| 가격 비교표 | 오토바이값, 배달 세팅 포함 가격 비교 |
| 중고 구매 체크리스트 | 구매 전 확인 항목 제공 |
| 초보 입문 가이드 | 면허, 보험, 안전, 운행 전 점검 안내 |

### 추가하면 좋은 기능

| 기능 | 설명 |
|---|---|
| 유지비 계산기 | 기름값, 오일, 타이어, 보험료 등 월 유지비 예상 |
| 질문 챗봇 | “PCX랑 NMAX 중 뭐가 나아요?” 같은 질문 답변 |
| 아버님 노하우 카드 | 실제 경험 기반 조언 저장 |
| 추천 결과 저장 | 추천 결과를 다시 볼 수 있게 저장 |
| 중고 매물 체크 도우미 | 사용자가 본 매물 정보를 입력하면 주의사항 제공 |

## 11. 초보 라이더 입문 가이드 구성

가이드 콘텐츠는 고정 페이지로 시작할 수 있다.

### 가이드 목차

| 단계 | 내용 |
|---|---|
| 1단계 | 면허 확인 |
| 2단계 | 보험 확인 |
| 3단계 | 오토바이 선택 |
| 4단계 | 배달 세팅 준비 |
| 5단계 | 운행 전 1분 점검 |
| 6단계 | 중고 구매 체크리스트 |
| 7단계 | 소모품 관리 |

## 23. 최종 정리

이 프로젝트는 단순 오토바이 추천 사이트가 아니다.

핵심은 다음과 같다.

```text
초보 라이더의 상황을 진단한다.
현실적인 대표 모델 4개만 비교한다.
가격과 운행 계획을 기준으로 추천한다.
추천 이유와 대안을 함께 제공한다.
중고 구매 실수를 줄인다.
배달 시작 전 기본 가이드를 제공한다.
아버님의 현장 경험을 지식 데이터로 반영한다.
```

따라서 최종 방향은 다음과 같다.

> **라이더 스타터 AI = 초보 배달 라이더를 위한 오토바이 선택 + 입문 가이드 + 구매 체크리스트 시스템**
